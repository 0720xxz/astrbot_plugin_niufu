# 🐮 通用服务器查询框架 for AstrBot

> 支持动态组别、多服务器状态查询、IP端口映射、黑名单管理、模糊提示等高级功能。

## ✨ 特色功能

- 🚀 **动态组别**：实时增删服务器组，无需重启机器人
- 📊 **在线人数查询**：'/查服 <组别名>' 获取指定组内所有服务器的在线人数（支持阿拉伯数字和中文数字自动排序）
- 🌐 **IP端口映射**：'/ip [组别名]' 获取服务器的公网连接地址
- 🛡️ **黑名单系统**：支持群聊/用户级别的黑名单过滤
- 🔧 **管理命令**：启停端口、修改服务器ID/名称、调整刷新间隔等
- 💡 **智能模糊提示**：当用户提到服务器名但未使用命令时，自动推送帮助信息
- 🔄 **后台缓存**：自动轮询刷新数据，降低API压力

## 📦 安装

1. 在 AstrBot 管理面板 -> 插件市场 -> 从 Git URL 安装：https://github.com/0720xxz/astrbot_plugin_niufu.git

2. 或手动下载解压到 'addons/astrbot_plugin_niufu' 目录，重启机器人。

## 🎮 使用方法

### 基础查询（所有人可用）

| 命令 | 说明 |
|------|------|
| '/查服 <组别名>' | 查看指定服务器组的在线人数（例如 '/查服 牛'） |
| '/ip [组别名]' | 查看指定组或所有服务器的 IP 和端口 |
| '/help' | 显示帮助信息 |

### 管理员命令（仅管理员）

| 命令 | 说明 |
|------|------|
| '/查看所有服' | 列出所有组别、服务器及启用状态 |
| '/添加服 <组别名> <识别名> <API_ID> <展示名>' | 添加新服务器 |
| '/删除服 <组别名> <识别名>' | 删除服务器 |
| '/启用端口 <所有/组别名>' 或 '/启用端口 <组别名> <识别名>' | 启用服务器(组) |
| '/禁用端口 <所有/组别名>' 或 '/禁用端口 <组别名> <识别名>' | 禁用服务器(组) |
| '/黑名单 <添加群/删除群/添加人/删除人> <号码>' | 群聊或用户级别黑名单 |
| '/设置组头部文字 <组别名> <第一行\|第二行\|分隔符>' | 自定义输出头部文字 |
| '/改服ID <组别名> <识别名> <新ID>' | 修改服务器API ID |
| '/改服名 <组别名> <识别名> <新展示名>' | 修改服务器展示名称 |
| '/改服组 <原组别名> <识别名> <新组别名>' | 将服务器移动到其他组 |
| '/调整刷新 <最小秒数> [最大秒数]' | 动态轮询间隔（最小2秒） |

## ⚙️ 配置文件

插件会在自身目录下自动生成以下配置文件：

- 'server_data.json' – 服务器组、服务器列表、头部文字等配置
- 'toggle_state.json' – 各服务器启用/禁用状态
- 'blacklist.json' – 黑名单列表

## 🌟 示例


/添加服 牛 饭堂1 69724 饭堂一服

/添加服 牛 饭堂2 69725 饭堂二服

/查服 牛


输出示例（按数字自动排序）


--- 牛 状态 ---
==============
饭堂一服 12/50
饭堂二服 8/50


## 🛠️ 注意

- 插件依赖 'aiohttp'，AstrBot 会自动安装。
- 管理员权限基于 AstrBot 根目录下 'data/cmd_config.json' 中的 'admins_id'。
- 默认的 API 地址为 'https://api.scplist.kr/api/servers/{id}'，如需修改可在代码中替换。
